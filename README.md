# 🏀 NBA Player Analytics Dashboard (Java Web Edition)

### 📌 專案簡介
本專案為 NBA 球員數據分析系統的 Java 實作版本。核心目標是將原先於 Python (Streamlit) 環境開發的數據分類邏輯，遷移至企業級的 Java Spring Boot 架構。系統支援 2000-2026 年跨時代球星之表現調閱，並提供動態分類模型分析與得分趨勢視覺化。

### 🛠️ 技術轉型與設計挑戰
* **開發框架**：核心後端採用 **Java Spring Boot 3**，前端結合 **Thymeleaf** 模板引擎與 **Bootstrap 5**。
* **數據獲取策略**：
    * **原技術規劃**：曾嘗試透過 Java `HttpClient` 與 `Jsoup` 實作即時網路爬蟲以抓取數據。
    * **當前實作**：由於 Java 爬蟲技術尚在研習階段，且為確保雲端環境（Render IP）存取之連線穩定性，本專案現階段採用**自建真實歷史數據庫（Mock Data）**進行功能演示。
* **核心運算邏輯**：
    * **活躍週期判定**：系統會自動驗證球員於選定賽季之活躍狀態。若球員已退役，系統將標註狀態並自動回溯顯示其生涯最後一季之表現數據。
    * **雙向連動控制**：實作了球隊與球員的 Cascading Select（連動下拉選單）邏輯。

### 📊 系統功能亮點
1.  **中文友善介面**：完整支援中文球隊名稱與分析標籤。
2.  **動態統計模型**：基於場均數據（PTS/REB/AST）進行即時分類。
3.  **趨勢分析圖表**：整合 **Chart.js** 動態繪製球員近期表現波動圖。
