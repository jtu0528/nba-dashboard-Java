## 📌 專案簡介
本專案為 NBA 球員數據分析系統的 Java 實作版本。核心目標是將原先於 Python (Streamlit) 環境開發的數據分類邏輯，遷移至企業級的 Java Spring Boot 架構。系統收錄了從 2000 年至 2025 年間 25 位跨時代頂級球星之歷史數據，並提供動態分類模型分析與多維度視覺化。

## 🛠️ 技術轉型與設計挑戰

* **開發框架：** 核心後端採用 Java Spring Boot 3，前端結合 Thymeleaf 模板引擎與 Bootstrap 5，並部署於 Render 雲端平台。
* **數據獲取策略：**
  * *原技術規劃：* 曾嘗試透過 Java `HttpClient` 與 `Jsoup` 實作即時網路爬蟲以抓取數據。
  * *當前實作：* 由於 Java 爬蟲技術尚在研習階段，且為確保雲端環境（Render IP 限制）存取之連線穩定性，本專案現階段採用**自建真實歷史數據庫**進行功能演示。
* **進階前端狀態管理 (State Management)：** * 重新設計了球隊與球員的 Cascading Select（連動下拉選單）邏輯。實作了「動作觸發器 (Action Trigger)」與「圖表釘選 (Pinned State)」，在切換球隊找尋新球員時，能保留當前觀測的球員圖表不破版。
  * 實作資料稀疏性防呆，當選擇的賽季中該球隊無收錄球員時，選單會動態提示 `-- 當季查無收錄球員 --`。

## 📊 系統功能亮點

* **中文友善介面：** 完整支援中文球隊名稱（跨越 24 支球隊歷史）與分析標籤。
* **深度歷史與交易追蹤：** 嚴格對齊球員在各個年份所屬的球隊。若球員經歷季中交易（如 2024 年 James Harden），系統會自動生成該季效力過的所有球隊切換按鈕（包含 TOT 賽季總計），提供無縫數據切換。
* **生涯活躍週期精準判定：** 系統會自動驗證球員於選定賽季之活躍狀態，動態渲染 `⏳ 尚未進入聯盟`、`🚨 賽季報銷 / DNP`。若球員已退役，系統將標註 `🏁 已退役` 並自動回溯顯示其生涯最後一季之表現數據。
* **動態統計模型與對比：** 基於場均數據進行即時球風分類（如：得分機器、組織大師、防守支柱），並自動計算該年度資料庫的真實平均值，繪製成球員表現與聯盟平均的柱狀對比圖。
* **多維度視覺化：** 整合 Chart.js 動態繪製球員五角能力分佈雷達圖，直觀呈現綜合實力。
